## Synopsis

```
var algebra = require('algebra');

var R = algebra.R;

var x = R(2),
    y = R(-1);

x.inv().mul(2).add(10).div(2);
```

## Installation

With [npm](https://npmjs.org/) do

```bash
$ npm install algebra
```
